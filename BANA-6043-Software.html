2 SAS

SAS is a commercial statistical computing software. It has been a prominent feature in analytics for decades. While R (and now Python) are gaining predominance for statistical computing, many large institutions continue to rely on SAS. In fact, I know of some large institutions that prohibit the use of R and Python, opting to strictly rely upon SAS.

SAS has some advantages. It is more of an “off-the-shelf” application than R or Python. While this limits flexibility, the methods to produce a scatterplot, for example, are straightforward and universal. The SAS language is not as complex as the languages of R or Python. In addition, everything in SAS is very well documented.

In this course, we will use SAS OnDemand for Academics. See http://support.sas.com/software/products/ondemand-academics/#s1=2 to get started. This is a convenient, cloud-based version of SAS that is free for students. In the workplace, you are much more likely to encounter the desktop version or Enterprise Guide. I will show some features of the desktop version in class that are different from the OnDemand.

3 R

R is an open source statistical computing software. This means that it is unlicensed and has many contributors. This can create obvious issues with replicability and versioning. For example, a program that relies upon a package that ceases to be compatible with later versions of R must find another solution.

R has some advantages, especially in contrast to SAS. Everything you can do in SAS you should be able to do in R. In addition, you should be able to do more in R than in SAS, generally speaking. R incorporates elements of general programming that allow you to create complex programs. However, tasks such as data manipulation and plotting are not as straightforward as they are in SAS.

A lot of firms have started relying more and more on R (and now Python) to capitalize on the flexibility of the R language. Even so, it is more likely that you will find resources for SAS at your workplace than you will find resources for R.

Follow this link to download the most recent version of R: https://cran.r-project.org/bin/windows/base/. Not all packages we use in this course may be compatible with prior versions. R 3.5 featured some significant updates.
