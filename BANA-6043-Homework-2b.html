The R code example below shows you how to add row and column names. For a correlation matrix, row and column names should be the same.

mat <- matrix(1, nrow=3, ncol=3)
# creates a 3x3 matrix

mat
##      [,1] [,2] [,3]
## [1,]    1    1    1
## [2,]    1    1    1
## [3,]    1    1    1
rn <- c('row1', 'var2', 'row3 or var 3')
# create vector of 3 for row names

rownames(mat) <- rn
# add row names to matrix

cn <- c('col1', 'col1', NA)
# create vector of 3 for column names

colnames(mat) <- cn
# add column names to matrix

mat
##               col1 col1 <NA>
## row1             1    1    1
## var2             1    1    1
## row3 or var 3    1    1    1
